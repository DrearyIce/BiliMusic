# BiliMusic - 哔哩音乐🎵
简洁的哔哩哔哩音乐客户端
## 注意 ⚠️
建议登录哔哩哔哩账号，否则大部分功能无法使用
编译完成后需要在 Data 文件夹中添加部分文件和文件夹，可以直接通过 Release 下载后添加到文件夹中。
应用内下载完成不会自动打开文件夹或有任何提示，请自行打开Data文件夹下的Downloads文件夹查看下载内容
当历史超过2时才会显示推荐
